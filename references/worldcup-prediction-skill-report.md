# worldcup-prediction Skill — 2026 世界杯预测系统

> 完整能力手册 · 2026 年 6 月

---

## 一、概述

**worldcup-prediction** 是一个面向 2026 年 FIFA 世界杯的端到端比赛预测与体彩推荐系统。它融合了统计建模、多源赔率数据、市场情绪分析和官方体彩数据，覆盖从单场比赛预测到整届赛事模拟的完整链路。

### 核心能力

- 基于 49,330 场历史国际比赛的 Elo 评分体系
- 多维度特征融合：Elo · FIFA 积分 · 球队身价 · 近期状态 · 阵容评分 · 赔率
- 确定性权重搜索 + 校准模型（非 AI 拍脑袋）
- 20,000 次蒙特卡洛淘汰赛模拟
- 双源赔率交叉验证（欧美庄家池 + 亚洲庄家池）
- 亚盘/大小球市场情绪分析
- 中国体彩推荐方案生成

## 三、预测方法详解

### 3.1 特征工程

**Elo 评分（核心强度特征）**

基于 49,330 场历史国际比赛（1872 年至今）构建：
- 初始值：1500
- K 因子动态调整（重要比赛权重更高）
- 主队优势：+60 Elo 点
- 最近场次窗口：10 场（用于近期状态）

**FIFA 积分**

57,794 行历史 FIFA 排名数据，覆盖所有国家队：
- 积分差值作为特征
- 排名差值作为辅助特征
- 与 Elo 形成独立的实力度量

**球队身价**

基于 Transfermarkt 数据的全队市场估值：
- 取对数比值（`log(主队身价 / 客队身价)`）
- 消除绝对数值差异，关注相对优势

**近期状态**

最近 10 场正式比赛滚动计算：
- 加权积分（近期比赛权重更高）
- 归一化为 -1 到 1 的标量

**阵容/战术画像**

基于游戏风格阵容评分构建的代理特征：
- 位置评分（门将/后卫/中场/前锋）
- 战术风格标注
- 明确标注为"代理评分"，不冒充官方数据

**赔率特征**

双源赔率共识，经去庄家水位处理：
```
赔率 → 隐含概率 → 去 margin → 归一化
```

### 3.2 模型结构

```
预测 = sigmoid(特征 × 权重)

其中特征向量：
  Elo 差值        经 sigmoid(Elo_diff / 420) 归一化
  FIFA 差值       经 sigmoid(FIFA_diff / 280) 归一化  
  身价比值        sigmoid(log_ratio)
  近期状态差值    经 sigmoid(form_diff × 2.2)

权重通过历史数据回测搜索确定：
  Elo       0.315  (主基线)
  主队优势   0.342
  平局偏向  -0.15
  平局倾向   1.413  (滚动统计)
```

**外部特征融合（当赛前数据可用时）：**

| 特征 | 权重 | 来源 |
|------|------|------|
| 赔率共识 | 0.35 | The Odds API + Titan007 |
| 阵容评分 | 0.08 | 游戏评级数据 |
| 身价 | 0.07 | Transfermarkt |

### 3.3 校准与验证

| 指标 | 训练期 (1994-2021) | 验证期 (2022-2026) |
|------|-------------------|-------------------|
| Log Loss | — | 0.873 |
| Brier Score | — | 0.513 |
| 准确率 | — | 59.99% |
| 样本量 | ~45,000 场 | ~4,000 场 |

校准使用 30 场滚动窗口 + 倾向特征微调。最新模型版本：
`elo-fifa-tendency-robust-modern-2026-05-20`

### 3.4 淘汰赛模拟

- 方法：蒙特卡洛模拟
- 迭代次数：20,000
- 小组赛：每组前两名晋级
- 淘汰赛：基于小组赛强度种子的启发式配对
- 每轮概率 = 小组赛强度 × 主场/中立调整

## 五、亚盘/大小球市场情绪分析

### 5.1 亚盘指标

从 Titan007 亚洲盘口页面提取每场比赛的庄家行为数据：

| 指标 | 含义 | 判断逻辑 |
|------|------|---------|
| 升盘公司数 | 庄家上调让球门槛 | 升盘数 >> 降盘数（显著多于）→ 看好让球方 |
| 降盘公司数 | 庄家下调让球门槛 | 降盘数 >> 升盘数（显著多于）→ 看衰让球方 |
| 高水公司数 | 让球方赔率高 | 低水数 >> 高水数（显著多于）→ 让球方安全 |
| 低水公司数 | 让球方赔率低 | 高水数 >> 低水数（显著多于）→ 让球方有阻力 |

> `>>` 符号在这里表示"远大于"、"显著多于"——当一方数量远超另一方时，才视为有效信号。例如"升盘 >> 降盘"=升盘公司数远多于降盘公司数，说明市场一致看好让球方。

## 六、体彩推荐系统

### 6.1 能力

- 自动对齐当前中国体彩 24 场世界杯体彩赛程与模型预测
- 基于赛事编号（周四001、周五003 等）精确定位
- 针对每场比赛提供多维度推荐

### 6.2 推荐维度

| 维度 | 条件 | 输出 |
|------|------|------|
| 单场胜负 | 模型概率 > 55% | 主胜/平局/客胜 + 概率 |
| 让球方向 | 双方实力差 > 30% | 强势方关注让球 |
| 比分波胆 | 模型预测明确 | 3 个最可能比分 |
| 过关串关 | 同一天 2+ 场高信心 | N 串 1 方案 |
| 市场提示 | AH/OU 信号非中性 | 升/降盘趋势 |

### 6.3 推荐示例

```
周四001 墨西哥 vs 南非   | 模型预测：胜72% 平17% 负12% | 比分2-0

推荐方案：
- 单场：主胜（72%）
- 让球：墨西哥占优
```

## 九、使用指南

### 9.1 日常流水线

```bash
# 1. 刷新赛前数据
node scripts/agent-preflight-update.mjs

# 2. 补充 Titan007 赔率
node scripts/import-titan007-odds.mjs

# 3. 批量预测 72 场
node scripts/batch-predict-2026.mjs

# 4. 淘汰赛模拟
node scripts/simulate-2026.mjs

# 5. 生成报告
node scripts/generate-report-2026.mjs

# 6. 亚盘大小球分析
node scripts/adjust-with-ah-ou.mjs

# 7. 体彩推荐
node scripts/fetch-jingcai-odds.mjs --only-worldcup
```

### 9.2 关键参数

```bash
# 单场比赛快速预测
node scripts/predict-match.mjs --home France --away Senegal --neutral true

# 权重搜索（重新训练）
node scripts/weight-search.mjs --data data/sample/matches.json --iterations 2000

# 按日期体彩推荐
node scripts/fetch-jingcai-odds.mjs --date=2026-06-11

# 控制台查看最新预测
cat output/world-cup-2026-report.md
cat output/ah-ou-market-report.md
cat output/jingcai-recommendations.md
```
